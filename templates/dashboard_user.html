Usage Examples

Use HTTP CONNECT proxy with curl, Python, or environment variables:

curl example:

curl -x http://proxy.bentoproxy.com \
--proxy-header "Proxy-Authorization: Basic $(echo -n 'api:{{ api_key }}' | base64)" \
https://ifconfig.me

Python requests example:

import requests

proxies = {
'http': 'http://api:{{ api_key }}@proxy.bentoproxy.com',
'https': 'http://api:{{ api_key }}@proxy.bentoproxy.com'
}

response = requests.get('https://ifconfig.me', proxies=proxies)
print(response.text) # Shows residential IP

Environment variables (works with most tools):

export http_proxy="http://api:{{ api_key }}@proxy.bentoproxy.com"
export https_proxy="http://api:{{ api_key }}@proxy.bentoproxy.com"

curl https://ifconfig.me # Now uses proxy automatically

Use SOCKS5 proxy with curl, Python, or any application that supports SOCKS5:

curl example:

curl --socks5-hostname {{ socks5_host }}:{{ socks5_port }} \
--proxy-user api:{{ api_key }} \
https://ifconfig.me

Python example:

import requests

proxies = {
'http': 'socks5://api:{{ api_key }}@{{ socks5_host }}:{{ socks5_port }}',
'https': 'socks5://api:{{ api_key }}@{{ socks5_host }}:{{ socks5_port }}'
}

response = requests.get('https://ifconfig.me', proxies=proxies)
print(response.text) # Shows residential IP
